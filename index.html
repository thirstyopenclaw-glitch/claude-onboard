OpenClaw // Claude Code Setup

Brand-new Mac. Do 3 short things — then Joel finishes the rest remotely.

Total hands-on time: ~3 minutes. After step 3 you're done — everything else (apps, tools, skills) gets installed for you over the network.

1Get online + install Tailscale

Connect to WiFi. Then open the App Store, search Tailscale, install it, open it, and sign in (any account is fine).

2Share this Mac with Joel

Go to login.tailscale.com/admin/machines, find this Mac, click the ⋯ → Share… and share it to:

thirstyopenclaw@gmail.com

3Run the bootstrap

This installs Joel's access key and turns on remote login. Two ways — pick one:

Easy: download it, then right-click the file → Open (right-click matters the first time, so macOS lets it run).

⬇ Download bootstrap.command

Or paste this in Terminal (⌘+Space → Terminal) — no download fuss:

curl -fsSL https://onboard.thirstyai.live/bootstrap.command -o ~/bootstrap.command && bash ~/bootstrap.command
It'll ask for your Mac password once (to turn on remote login) — normal, you won't see it as you type.

4Text Joel "ready"

The bootstrap prints a line like ssh you@your-ip. Send that to Joel. He connects and installs everything else — Ghostty, Claude Code, your skills — remotely. You're done. ✅
