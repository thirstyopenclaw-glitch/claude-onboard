OpenClaw // Claude Code Setup

Brand-new Mac to fully running. Do these in order. Tap COPY, paste in the terminal, hit Enter.

Before you start: Connect to WiFi. This takes about 20-30 minutes, mostly waiting on downloads — that's normal, don't close anything.
Needs macOS 13 or newer (any Mac from the last few years).
Open Terminal now: press ⌘ + Space, type Terminal, Enter. Keep this page open beside it.

1Install Homebrew

The installer for everything else. Paste, Enter, type your Mac password if asked (you won't see it as you type — that's normal, just type and Enter).

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
A box may pop up asking to install "Command Line Developer Tools" — click Install and wait (5-15 min). This is required. When done, run the Homebrew command again if it stopped.

At the very end Homebrew prints two lines starting with echo (an "Add Homebrew to your PATH" step). Copy and run both — otherwise brew won't work.

2Install Ghostty (your terminal)

A nicer, faster terminal. After it installs, quit Terminal and open Ghostty (⌘ + Space → Ghostty) and do the rest of the steps there.

brew install --cask ghostty

3Install Claude Code

The main tool. Run this in Ghostty.

curl -fsSL https://claude.ai/install.sh | bash
If you later type claude and it says "command not found": fully quit Ghostty (⌘Q) and reopen it, then try again. (It installs to ~/.local/bin and needs a fresh window.)

Check it worked:

claude doctor

4Start Claude Code & sign in

Run this — a browser window opens. Log in and approve.

claude
You need a PAID plan — Claude Pro ($20/mo) or Max. The free Claude account will not work here. Get one at claude.com before this step.

5Install Tailscale (so Joel can help remotely)

A private network that lets Joel connect to finish your setup.

brew install --cask tailscale

Open Tailscale (⌘ + Space → Tailscale) and sign in.

Important: Joel and you must be on the same Tailscale network. Either sign in with the account Joel gives you, or sign in with your own and have Joel send you an invite link to join his. Text Joel once you're signed in and he'll confirm he can see your machine.

6Turn on Remote Login (SSH)

Lets Joel connect over Tailscale. Do this by hand in Settings:

System Settings → General → Sharing → turn ON "Remote Login"

It will show a line like ssh youraccount@your-mac. Text that whole line to Joel.

7Starter skill pack (optional)

Adds the basics: research agent, simple archive, save-session. Paste in Ghostty.

curl -fsSL https://onboard.thirstyai.live/starter-pack.sh | bash
If this gives a "404" or error, the pack isn't published yet — just skip it. Joel installs it for you over Tailscale.

8You're set — try it

In Claude Code, just talk to it like a person, or type a slash command:

/research what's the weather in Park City today

That's the whole setup. Text Joel "done" and he'll finish the rest remotely.
